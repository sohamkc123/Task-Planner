# jQuery Task Planner — Soham KC

A simple and interactive Task Planner Web App built as a jQuery lab practical project.

This project demonstrates important jQuery concepts like DOM manipulation, animations, AJAX, selectors, and form validation in a clean and practical way.

## Features / Programs Covered

| # | Program | Where it is |
|---|---------|-------------|
| 1 | mouseenter / mouseleave | Hover over profile photo in sidebar |
| 2 | AJAX | "Load Quote" button in sidebar |
| 3 | Method Chaining | Throughout — fadeIn, slideDown, animate |
| 4 | Show / Hide | Registration panel opens/closes |
| 5 | jQuery Selectors | Filter buttons on task list |
| 6 | Form Validation | Registration slide panel |
